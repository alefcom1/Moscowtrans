# Руководство по генерации страниц — Бюро переводов «Ремарка»

## Общий контекст

**Бюро переводов «Ремарка»**, Москва. Только B2B-переводы.  
Никаких: нотариального заверения, апостиля, паспортов, свидетельств ЗАГС, личных документов физических лиц.  
Фокус: корпоративные и технические переводы для юридических лиц и индивидуальных предпринимателей.

**Технический стек:** статический HTML, тёмная glassmorphism-тема (`data-theme="dark"/"light"` на `<html>`).  
Акцент: `#783CF0` → `#00A0F0`. Шрифты: Inter + Georgia.  
**Никогда не упоминать Краснодар** — только Москва и Россия в целом.

# Инструкция по генерации одной страницы

1. Определить раздел (технический/медицинский/юридический/IT) и тип (отрасль/документ).
2. Взять `tekhnicheskiy-perevod.html` (или соответствующий хаб-файл) как основу.
3. Заменить `<title>`, `<meta name="description">`, `<link rel="canonical">`, JSON-LD.
4. Обновить хлебные крошки: `Главная / [Хаб] / [Название страницы]`.
5. Заменить H1 и intro-текст; адаптировать 4 блока `intro-feat` под специфику темы.
6. В `sec-docs` заменить категории документов на релевантные для данной отрасли/типа.
7. **Добавить `sec-audience` (секция «Для кого»)** после `sec-docs`: 5–7 карточек B2B-аудитории из раздела «Аудитория по разделам» выше. CSS-класс `doc-ref-grid` / `doc-ref-card` — уже в `styles.css`.
8. В `sec-translators` адаптировать специализацию переводчиков под тему.
